# Install

```sh
# run install script
./install.sh

# install formatters : xmllint and jq
dnf install  xmllint  jq

# for better usage also install
sd          # search and replace
snippets-ls # snippets lsp (https://github.com/mgholam/snippets-ls)
tmux
lazygit + delta # side by side diff
uncrustify  # c# formatting
biome       # js/html/css formatting
```

## update

```sh
# view updates available
npm outdated -g 
# update
npm update -g
```
